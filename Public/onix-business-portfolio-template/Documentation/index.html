B) CSS Files and Structure - top

There are three css files necessary for customization.

  1. "assets/css/style.css" -Is the main css file.
  2. "assets/css/reset.css" -Is the reset html css file.
  3. "assets/css/prettyPhoto.css" -Is the prettyPhoto css file.
  4. "assets/css/responsiveslides.css.css" -Is the rslider css file.

    5. The style.css file is separated sections:
